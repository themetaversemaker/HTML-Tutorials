The mark tag is highlight a particular word

The HTML del element defines text that has been deleted from a document. Browsers will usually strike a line through deleted text

The HTML ins element defines a text that has been inserted into a document. Browsers will usually underline inserted text

The Subscript text appears half a character below the normal line, and is usually rendered in a smaller font - Example :: H2O

The Superscript text appears half a character above the normal line, and is usually rendered in a smaller font - Example :: 22 = 4
